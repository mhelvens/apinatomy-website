ApiNATOMY Prototype Application

A prototype of the ApiNATOMY circuitboard experience, including cross-tile connectors, 3D models, time-variant models and more is running here. It is a web-application built with, among other things, jQuery, AngularJS, D3.js and three.js.

ApiNATOMY Core Widget

Work is being done on a core module for the ApiNATOMY circuitboard. It is domain-agnostic, and can therefore be used to display, navigate and edit any ontology. It has no dependencies other than jQuery. In particular, AngularJS is not required, to it can be used by a wider audience.

The ApiNATOMY prototype application listed above will be refactored to use this module when a certain level of maturity is reached.
